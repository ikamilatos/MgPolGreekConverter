# MgPolGreekConverter

Convert Magenta Polytonic Greek Word 6 files to DOCX Unicode.

The process goes as follow:
 * Open the word file using Microsft Word with DOCX support and save the file as a DOCX file.
 * Then run the program and will create a file with the _fixed.docx extension that will contain the converted file.
 * Once you are satisfied with the convertion can delete the intermediate docx file.

.NET 10 is used and opens the DOCX files using Zip and XML approach.
